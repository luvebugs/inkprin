**Prompt for AI Tattoo Design Generation:**

---

**Task:** Create a unique tattoo design based on the uploaded image or provided text input. The output should include a visual representation of the tattoo design as well as an example of how it would appear on human skin.

**Inputs:**
1. **Image Input (optional):** Upload an image that inspires the tattoo design. This can be anything from a simple shape, an animal, a symbol, or an intricate artwork.
2. **Text Input (optional):** Provide descriptive text or keywords that convey the desired theme, style, or message of the tattoo. This could include specific elements like "tribal," "floral," "geometric," "quote," etc.

**Outputs:**
1. **Tattoo Design:** Generate a detailed and original tattoo design based on the image and/or text input. Ensure the design is clear, aesthetically pleasing, and suitable for tattooing.
2. **Skin Visualization:** Create a realistic mockup showcasing how the tattoo design would look when applied to human skin. Include variations in skin tones and placements (e.g., arm, back, leg) to provide a comprehensive view.

**Constraints:**
- The design must be original and not copied from existing works.
- Ensure that the tattoo design adheres to common tattooing practices and aesthetics.
- The skin visualization should accurately represent the scale and details of the tattoo on human skin.

**Instructions:**
1. Analyze the uploaded image and/or text input for thematic elements and stylistic preferences.
2. Synthesize these elements into a cohesive tattoo design.
3. Generate a high-resolution image of the tattoo design.
4. Create a mockup image displaying the tattoo on a model's skin, considering realistic shading and color application.

**End of Prompt**
